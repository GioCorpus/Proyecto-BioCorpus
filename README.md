Proyecto: BioCorpus

Plataforma de bioinformática para descubrimiento de vacunas usando datos genómicos, IA y análisis de proteínas.

Objetivo:

Analizar genomas de patógenos y sugerir candidatos de antígenos para el desarrollo de vacunas.

Lenguaje principal:

Python

Librerías clave:

Biopython

PyTorch

TensorFlow

AlphaFold

BLAST

Bases de datos:

National Center for Biotechnology Information

UniProt Consortium

Protein Data Bank
